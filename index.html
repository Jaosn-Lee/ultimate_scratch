What is scratch (for newbies)

If you don't know, scratch is a visual block programming language made by MIT, it uses simple blocks to code and it's mainly for kids and people who are completely new at programming. Me personally, I used scratch a lot when I was like 9 or 10, and now looking back on it, I probally wouldn't have gotten into coding or programming if it wasn't for Scratch!
